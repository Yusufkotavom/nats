# Mulai di Sini

Panduan ini ditujukan untuk user operasional dan admin bisnis, bukan developer. Semua dokumentasi telah diperbaharui dengan **step-by-step detail** dan **contoh praktis**.

## 🚀 Urutan Belajar Disarankan

### 1. Setup & Persiapan
1. **[Setup Awal](./01-setup-awal)** - Panduan lengkap setup sistem dari nol (3-4 jam)
2. **[Master Data](./02-master-data)** - Input semua data fondasi dengan contoh detail
3. **[Operasional Harian](./03-operasional-harian)** - Workflow standar restoran

### 2. Panduan Operasional
4. **[Panduan Cepat Harian](./quick-daily-operations)** - Checklist operasional lengkap
5. **[Use Case Lengkap](./complete-use-cases)** - Skenario nyata dari setup sampai troubleshooting
6. **Panduan per modul** pada folder `modules/` (semua sudah diperbaharui)

## 📋 Modul yang Didukung

### Modul Inti Operasional
- **[POS](./modules/pos)** - Kasir dan transaksi penjualan (panduan detail 40+ langkah)
- **[Sales](./modules/sales)** - Manajemen penjualan B2B dan catering (workflow lengkap)
- **[Purchase](./modules/purchase)** - Pembelian dari PO sampai payment (contoh praktis)
- **[Inventory](./modules/inventory)** - Stok barang dan pergerakan (tips harian)
- **[Production](./modules/production)** - BOM dan produksi (resep detail + cost analysis)

### Modul Keuangan
- **[Cash & Bank](./modules/cash-bank)** - Manajemen kas dan rekening (workflow praktis)
- **[Accounting](./modules/accounting)** - Jurnal dan laporan keuangan (untuk awam akuntansi)
- **[Accounting Best Practices](./modules/accounting-best-practices)** - Panduan khusus restoran

### Modul Pendukung
- **Budgeting** - Perencanaan anggaran
- **Assets** - Manajemen aset tetap
- **HR** - Sumber daya manusia
- **AI** - Fitur kecerdasan buatan
- **Admin** - Pengaturan sistem

## 🎯 Prinsip Operasional

- **Data First**: Jangan transaksi sebelum master data inti lengkap
- **Step by Step**: Ikuti urutan setup, jangan lompat langkah
- **Test Everything**: Selalu test dengan data kecil dulu
- **Backup Regular**: Selalu backup data dan cocokkan kas fisik dengan sistem
- **User Friendly**: Semua panduan dibuat untuk non-technical users
