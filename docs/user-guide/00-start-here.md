# Mulai di Sini

Panduan ini ditujukan untuk user operasional dan admin bisnis, bukan developer.

## Urutan Belajar Disarankan

1. Baca [Setup Awal](./01-setup-awal) untuk akses sistem.
2. Lanjut [Master Data](./02-master-data) untuk data fondasi.
3. Jalankan [Operasional Harian](./03-operasional-harian).
4. Gunakan panduan per modul pada folder `modules/`.

## Modul yang Didukung

- POS
- Sales
- Purchase
- Cash & Bank
- Inventory
- Production
- Accounting
- Budgeting
- Assets
- HR
- AI
- Admin

## Prinsip Operasional

- Jangan transaksi sebelum master data inti lengkap.
- Untuk restoran: stok awal minimal bisa `0`, lalu stok bertambah dari transaksi masuk.
- BOM dipakai agar penjualan menu mengurangi bahan baku secara otomatis.
