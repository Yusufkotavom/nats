# Modul Production

## Fungsi
- BOM
- Production Order
- Production Issue
- Production Receipt

## Untuk Restoran
- BOM dipakai sebagai resep menu.
- Pastikan qty BOM sesuai base unit bahan.

## Validasi
- Issue mengurangi stok bahan.
- Receipt menambah stok produk jadi (jika dipakai).
