# Glosarium

- **BOM**: Resep/komposisi bahan untuk menghasilkan 1 unit produk menu.
- **Base Unit**: Satuan dasar stok dan biaya per produk.
- **Conversion Factor**: Faktor konversi antar unit pada level produk.
- **POS Session**: Sesi kasir dari buka sampai tutup kas.
- **Inventory Movement**: Catatan masuk/keluar/adjustment stok.
- **COGS**: Harga pokok penjualan.
- **Outbox**: Antrean event integrasi sistem.
